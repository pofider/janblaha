Current projects

jsreport.net

jsreport is an open source reporting platform dedicated to developers who whants to design and render business reports. It is mainly based on javascript templating engines and html to pdf conversion using phantomjs. It's distributed as free on premise installation or cloud based SaaS service.

mobilefieldreport.com

Mobile Field Report is a SAS project helping mobile workers in the field. It consists of html based portal and iPad client. It is supporting advanced features like job scheduling, building tree of job tasks, time tracking or offline client synchronizing. Solution is build on .NET and hosted in Windows Azure. Application is highly scalable using Azure cloud services and SQL database federation.

postuj.cz

Postuj.cz is Czech community social project joining diferent people interested in specific location and specific category. Project is implemented as SPA using .NET and Windows Azure hosting.
